如何用20G过完一个月

好像之前有说过要写一篇blog来说明一下如何搞才能用20G愉快的过完一个月。。所以现在就来了 Orz

大家要先了解一下清华的网络收费规则，很重要的一点就是清华的网络支持IPv6，而且IPv6是不收费的。。。so，大家就已经就明白要怎么搞了对吧。。

STEP 1 申请一个支持IPv6的VPS

首先你需要有一台支持IPv6的服务器，这里比较推荐DigitalOcean的在旧金山的VPS；网络延迟比较小而且支持IPV6。

但是！这个VPS一个月要5美元！简直丧心病狂。。难道我们要为了剩下几十块的网费而花几十块来买服务器么。。当然不是。。这里推荐两种方法来省钱。

第一种是为程序猴子们准备的（程序猴子已经不需要看这个了吧）；可以尝试申请github的education pack，然后里边有digitalocean的100美元。但是需要注意，这里是100$ for new user，所以应该是先申请这个pack然后再去申请VPS。

第二种就是大家可以用别人的优惠代码，免费代码可以当10美元用。我的优惠连接是这个，大家可以拿去用嗯。

这两种方法都需要先充值5美元才能建立VPS。这里可以使用paypal付款。

然后新建一个VPS，开通IPv6就行了。

STEP 2 配置shadowsocks

接下来就要搭建我们的VPN了！这里还是推荐shadowsocks，具体配置方法见这里。

另外像那上边开启服务器比较蛋疼，所以可以这么做：

首先创建一个文件：/etc/shadowsocks.json，示例如下：

{
    "server":"::",
    "server_port":8388,
    "local_address": "127.0.0.1",
    "local_port":1080,
    "password":"你设置的密码",
    "timeout":300,
    "method":"aes-256-cfb",
    "fast_open": false
}

接下来你就可以使用下面这个指令启动服务:

ssserver -c /etc/shadowsocks.json -d start

停止指令：

ssserver -c /etc/shadowsocks.json -d stop

很简单对吧。。

STEP 3 配置客户端

  • OSX / Windows
  • Android / IOS

然后比较蛋疼的就是，上述的链接是被墙掉的。。所以想下载的小伙伴请联系我或者找能到墙外的小伙伴帮忙把 OwO

在客户端上（Window为例子），新建一个配置，IP地址写STRP 1中得到的IPv6地址，然后打开全局代理，就能愉快的免流量上网了 OwO 。建议再新建一个服务器配置，里边写你的服务器的IPv4地址，这样就能在不支持IPv6的网络环境中愉快的上网了！
